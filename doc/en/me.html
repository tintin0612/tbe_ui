🧸 TbeUI 🌸 ​

Keep interfaces soft and gentle, write warmth into code

A neumorphic CSS component library filled with gentle textures

If you like it, buy me a cup of milk tea~

✨ Features ​

🎯 60+ Beautiful Components - Buttons, cards, forms, navigation and more 🌙 Dark Mode Support - Gentle visual experience that's easy on the eyes 📱 Responsive Design - Looks great on mobile and tablet 🎨 Exquisite Texture - Soft silicone touch feeling ♿ Accessibility Support - Usable by everyone

🚀 Quick Start ​

📦 Installation ​

bash
npm install tbe-ui

✨ Basic Usage ​

html
<!-- Import styles -->
<link rel="stylesheet" href="node_modules/tbe-ui/dist/tbeui.css">

<!-- Import JavaScript -->
<script src="node_modules/tbe-ui/dist/tbeui.umd.js"></script>
<script>
  // Initialize components
  TbeUI.init();
</script>

🎯 Using Components ​

html
<!-- Buttons - Soft and squishy -->
<button class="t-button">Default Button</button>
<button class="t-button t-button--primary">Primary Button</button>

<!-- Card - Contains little surprises -->
<div class="t-card">
  <h3>🎀 A little card for you</h3>
  <p>Contains sweet words~</p>
</div>

<!-- Input - Write your little secrets -->
<input class="t-input" placeholder="✍️ Write something here...">
</script>

📖 Let me tell you a story... ​

When I was little, there was a 💗 silicone calculator at home. The buttons were soft and squishy, with a springy feedback when pressed. Later, these calculators gradually disappeared, and that warm tactile feeling was nowhere to be found. Until recently, I unexpectedly found the same model on Taobao! At that moment, all my childhood memories came flooding back...🥹

So I had this idea: turn this gentle, cute, retro silicone texture into a UI component library. Let code have warmth, let interfaces convey warmth.

As someone who loves frontend development, over these past two months, I've written and rewritten, revised and revised again. During this time, I've truly sought help from many enthusiastic experts who patiently helped me review code and answer questions...
