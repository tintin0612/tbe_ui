💡 Usage Tips ​

Features ​

  1. Real-time Preview: The preview updates automatically after editing HTML code
  2. Code Copy: Click the "Copy Code" button to copy current code to clipboard
  3. Reset Function: Click the "Reset" button to restore initial code
  4. Tab Switching: Toggle between HTML editing and preview

Common Operations ​

  • Edit Code: Edit code in the HTML tab
  • View Results: Switch to preview tab to see results
  • Copy Code: Copy code for use in your project
  • Adjust Code: Modify and adjust component properties as needed

Notes ​

  • Playground supports all Tbe UI components
  • Ensure TbeUI.init() is properly included in your code
  • Some advanced features may require additional JavaScript code
  • Complex components may need more configuration and initialization code
