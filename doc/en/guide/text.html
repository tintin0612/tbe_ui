📝 Text Styles ​

Tbe UI provides a complete set of text style utilities for quickly controlling text alignment, color, size, weight, and other effects.

Left-aligned text. This is a left-aligned example text used to demonstrate text alignment effects.

t-ta-c

Center-aligned text. This is a center-aligned example text used to demonstrate text alignment effects.

t-ta-r

Right-aligned text. This is a right-aligned example text used to demonstrate text alignment effects.

html

Line Height ​

t-lh-tight (1.25)

Tight line height. This text uses tight line height settings. Suitable for titles or compact typography scenarios.

t-lh-normal (1.5)

Normal line height. This text uses normal line height settings. Suitable for most body text.

t-lh-loose (2)

Loose line height. This text uses loose line height settings. Suitable for scenarios requiring larger reading space.

html
<p class="t-lh-tight">Tight line height text</p>
<p class="t-lh-normal">Normal line height text</p>
<p class="t-lh-loose">Loose line height text</p>

Text Overflow ​

Single line ellipsis t-to-e
This is a very long text content that will be truncated and show an ellipsis when exceeding container width.
Multi-line ellipsis
This is a very long multi-line text content that can display two lines. When text exceeds the specified number of lines, it will be truncated and show an ellipsis. This is very useful for card titles or summary previews, suitable for various scenarios that need to limit display lines.
html
<p class="t-to-e t-wh-n t-w-200">Single line ellipsis text</p>
<p class="t-to-m2 t-wh-nw t-lh-3 t-h-80">Multi-line ellipsis text</p>
