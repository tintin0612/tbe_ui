📐 Sizing ​

Tbe UI provides a complete set of sizing utilities for quickly controlling element width, height, min/max values, and more.

t-w-auto
t-w-screen (100vw)
html
<div class="t-w-full">Width 100%</div>
<div class="t-w-auto">Width auto</div>
<div class="t-w-screen">Width 100vw</div>
