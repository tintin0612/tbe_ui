🛠️ Utilities Reference ​

Tbe UI provides a complete set of utility classes covering spacing, layout, text, positioning, and other common scenarios. All utility classes use !important declarations and can directly override component default styles.

Style Override Priority ​

Utility classes use !important declarations to ensure they can override component default styles:

css
/* Utility classes always take priority */
.t-card {
  padding: 24px; /* Component default */
}
.t-p-10 {
  padding: 10px !important; /* Utility class takes priority */
}
