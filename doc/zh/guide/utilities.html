🛠️ Utilities 工具类参考 ​

Tbe UI 提供了一套完整的工具类，覆盖间距、布局、文本、定位等常用场景。所有工具类都使用 !important 声明，可以直接覆盖组件默认样式。

快速导航 ​

  • 间距 Spacing - padding / margin
  • 弹性布局 Flexbox - flex 容器 / 对齐 / 间距
  • 文本样式 Text - 文字 / 颜色 / 溢出
  • 定位布局 Position - 定位 / 浮动 / 显示 / z-index
  • 尺寸 Sizing - 宽度 / 高度 / 最小最大

样式覆盖优先级 ​

工具类使用 !important 声明，确保可以覆盖组件默认样式：

css
/* 工具类始终优先 */
.t-card {
  padding: 24px; /* 组件默认 */
}
.t-p-10 {
  padding: 10px !important; /* 工具类优先 */
}

响应式前缀 ​

前缀断点说明
t-xs:0px+超小屏幕
t-sm:≥576px小屏幕
t-md:≥768px中等屏幕
t-lg:≥992px大屏幕
t-xl:≥1200px超大屏幕
t-xxl:≥1400px极大屏幕
html
<!-- 响应式示例 -->
<div class="t-w-full t-md-w-50 t-lg-w-33">
  移动端全宽，桌面端半宽，更大屏幕三分之一宽
</div>
