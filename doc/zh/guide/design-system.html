设计系统 ​

Tbe UI 提供了一套完整的设计系统，包含色彩、排版、间距、边框等设计元素，帮助你构建一致性的用户界面。

🎨 设计原则 ​

Tbe UI 的设计系统遵循以下核心原则：

  • 一致性：统一的设计语言和视觉规范
  • 可访问性：确保所有用户都能舒适使用
  • 灵活性：支持主题定制和个性化配置
  • 性能优化：轻量化设计，优秀的加载性能

🚀 主题定制 ​

CSS 变量覆盖 ​

css
:root {
  /* 自定义主题色 */
  --t-color-primary: #1890ff;
  --t-color-success: #52c41a;
  --t-color-warning: #faad14;
  --t-color-danger: #f5222d;
  
  /* 自定义文字色 */
  --t-text-color-primary: #262626;
  
  /* 自定义背景色 */
  --t-bg-color: #fafafa;
}

深色模式 ​

css
/* 深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --t-bg-color: #141414;
    --t-text-color-primary: #e8e8e8;
    --t-border-color: #434343;
  }
}

💡 最佳实践 ​

  1. 保持一致性：在整个应用中使用相同的设计系统
  2. 遵循层级：使用适当的字体大小和颜色区分内容重要性
  3. 考虑可访问性：确保颜色对比度符合 WCAG 标准
  4. 响应式设计：使用相对单位和响应式工具类
  5. 性能优化：优先使用 CSS 变量和工具类，减少重复代码

通过遵循 Tbe UI 的设计系统，你可以创建出视觉一致、用户友好的界面设计。
