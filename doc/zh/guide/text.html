📝 Text 文本样式 ​

Tbe UI 提供了一套完整的文本样式工具类，用于快速控制文本的对齐、颜色、大小、粗细等效果。

文本对齐 ​

t-ta-l

左对齐文本。这是一段左对齐的示例文本，用于展示文本对齐效果。

t-ta-c

居中对齐文本。这是一段居中对齐的示例文本，用于展示文本对齐效果。

t-ta-r

右对齐文本。这是一段右对齐的示例文本，用于展示文本对齐效果。

html
<p class="t-ta-l">左对齐文本</p>
<p class="t-ta-c">居中对齐文本</p>
<p class="t-ta-r">右对齐文本</p>

行高 (Line Height) ​

t-lh-tight (1.25)

紧凑行高。这段文本使用了紧凑的行高设置。适用于标题或需要紧凑排版的场景。

t-lh-normal (1.5)

正常行高。这段文本使用了正常的行高设置。适用于大多数正文文本。

t-lh-loose (2)

宽松行高。这段文本使用了宽松的行高设置。适用于需要更大阅读空间的场景。

html
<p class="t-lh-tight">紧凑行高文本</p>
<p class="t-lh-normal">正常行高文本</p>
<p class="t-lh-loose">宽松行高文本</p>

文本溢出 ​

单行省略 t-to-e
这是一段很长的文本内容，超出容器宽度后会被截断显示省略号。
多行省略
这是一段很长的多行文本内容，可以显示两行。当文本超过指定的行数后，会被截断并显示省略号。这在卡片标题或摘要预览中非常有用，适用于各种需要限制显示行数的场景。
html
<p class="t-to-e t-wh-n t-w-200">单行省略号文本</p>
<p class="t-to-m2 t-wh-nw t-lh-3 t-h-80">多行省略号文本</p>

常用组合 ​

卡片标题 (t-fs-20 t-fw-b)

这是一段正文描述文本，使用了次要文字颜色和正常行高。适用于卡片的说明文字或摘要内容。

标签一标签二2024-01-01
html
<h3 class="t-fs-20 t-fw-b t-m-0 t-mb-10">卡片标题</h3>
<p class="t-text-sm t-c-text-secondary t-lh-normal">正文描述</p>
<span class="t-tag t-tag-primary">标签</span>
<span class="t-c-text-placeholder t-text-sm t-ml-auto">日期</span>
