Upload 上传 ​

Upload 是一种文件上传组件，支持多文件上传、拖拽上传等功能。

基础上传 ​

最简单的文件上传。

只能上传jpg/png文件，且不超过500kb
    html
    <div class="t-upload" id="upload1" data-auto-upload="false">
  <div class="t-upload__trigger">
    <button class="t-btn t-btn--primary" onclick="document.getElementById('upload1-input').click()">
      <i class="t-icon-upload"></i> 点击上传
    </button>
    <input type="file" id="upload1-input" class="t-upload__input" style="display: none;" onchange="handleFileSelect('upload1', this.files)">
  </div>
  <div class="t-upload__tip">只能上传jpg/png文件，且不超过500kb</div>
  <ul class="t-upload__file-list"></ul>
</div>

    拖拽上传 ​

    支持拖拽上传文件。

    点击或拖拽文件到此处上传
    只能上传jpg/png文件，且不超过500kb
      html
      <div class="t-upload t-upload--drag" id="upload2" data-auto-upload="false">
  <div class="t-upload__drag-area" onclick="document.getElementById('upload2-input').click()" ondrop="handleDrop(event, 'upload2')" ondragover="handleDragOver(event)" ondragleave="handleDragLeave(event)">
    <i class="t-icon-upload" style="font-size: 48px;"></i>
    <div class="t-upload__text">点击或拖拽文件到此处上传</div>
    <div class="t-upload__tip">只能上传jpg/png文件，且不超过500kb</div>
    <input type="file" id="upload2-input" class="t-upload__input" style="display: none;" onchange="handleFileSelect('upload2', this.files)">
  </div>
  <ul class="t-upload__file-list"></ul>
</div>

      图片缩略图上传 ​

      上传图片时显示缩略图预览。

      支持多图上传，点击开始上传模拟上传过程
      html
      <div class="t-upload" id="upload3" data-auto-upload="false">
  <div class="t-upload__trigger">
    <button class="t-btn t-btn--primary" onclick="document.getElementById('upload3-input').click()">
      <i class="t-icon-picture"></i> 选择图片
    </button>
    <button class="t-btn t-btn--success" onclick="simulateUpload('upload3')">开始上传</button>
    <input type="file" id="upload3-input" class="t-upload__input" accept="image/*" multiple style="display: none;" onchange="handleImageSelect('upload3', this.files)">
  </div>
  <div class="t-upload__tip">支持多图上传，点击开始上传模拟上传过程</div>
  <div class="t-upload__picture-list" id="upload3-picture-list"></div>
</div>

      虚拟上传演示 ​

      模拟真实上传过程，展示进度条和状态变化。

      选择文件后点击"模拟上传"查看上传进度
        html
        <div class="t-upload" id="upload5" data-auto-upload="false">
  <div class="t-upload__trigger">
    <button class="t-btn t-btn--primary" onclick="document.getElementById('upload5-input').click()">
      <i class="t-icon-upload"></i> 选择文件
    </button>
    <button class="t-btn t-btn--success" onclick="simulateUploadWithProgress('upload5')">模拟上传</button>
    <button class="t-btn" onclick="clearUpload('upload5')">清空列表</button>
    <input type="file" id="upload5-input" class="t-upload__input" multiple style="display: none;" onchange="handleFileSelectWithPreview('upload5', this.files)">
  </div>
  <div class="t-upload__tip">选择文件后点击"模拟上传"查看上传进度</div>
  <ul class="t-upload__file-list" id="upload5-list"></ul>
</div>

        API ​

        属性 ​

        属性说明类型可选值默认值
        action上传地址string——
        auto-upload是否自动上传booleantrue/falsetrue
        multiple是否支持多文件上传booleantrue/falsefalse
        accept接受的文件类型string——
        drag是否支持拖拽上传booleantrue/falsefalse

        事件 ​

        事件名说明回调参数
        change文件选择改变时触发files
        success上传成功时触发response, file
        error上传失败时触发error, file
        progress上传进度变化时触发progress, file
        remove文件移除时触发file

        方法 ​

        方法名说明参数
        submit手动上传—
        clearFiles清空文件列表—
        abort取消上传file
