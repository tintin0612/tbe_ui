Popover 弹出框 ​

Popover 是一种轻量级的弹出框，用于显示提示或附加信息。

基础用法 ​

四个方向 ​

点击按钮触发弹出框，可以设置弹出框出现在上下左右四个方向。

自定义内容 ​

操作按钮 ​

在弹出框中放置操作按钮。

确认操作

确定要删除这条记录吗？

html
<div class="t-popover t-popover-top" id="popover7">
  <button class="t-btn" onclick="toggleTPopover('popover7')">操作按钮</button>
  <div class="t-popover-content">
    <div class="t-popover-title">确认操作</div>
    <div class="t-popover-body">
      <p>确定要删除这条记录吗？</p>
      <div class="flex-gap" style="margin-top: 12px; justify-content: flex-end;">
        <button class="t-btn t-btn-sm" onclick="closeTPopover('popover7')">取消</button>
        <button class="t-btn t-btn-sm t-btn-danger" onclick="TMessage.success('删除成功'); closeTPopover('popover7')">确定</button>
      </div>
    </div>
  </div>
</div>

表单内容 ​

在弹出框中放置表单内容。

用户信息
html
<div class="t-popover t-popover-bottom" id="popover8">
  <button class="t-btn" onclick="toggleTPopover('popover8')">表单内容</button>
  <div class="t-popover-content" style="width: 240px;">
    <div class="t-popover-title">用户信息</div>
    <div class="t-popover-body">
      <div class="t-form-item" style="margin-bottom: 12px;">
        <label class="t-form-label" style="font-size: 12px;">用户名</label>
        <input type="text" class="t-input t-p-5" placeholder="请输入用户名" style="font-size: 12px; padding: 6px 10px;">
      </div>
      <div class="t-form-item" style="margin-bottom: 12px;">
        <label class="t-form-label" style="font-size: 12px;">邮箱</label>
        <input type="email" class="t-input" placeholder="请输入邮箱" style="font-size: 12px; padding: 6px 10px;">
      </div>
      <div class="flex-gap" style="justify-content: flex-end;">
        <button class="t-btn t-btn-sm" onclick="closeTPopover('popover8')">取消</button>
        <button class="t-btn t-btn-sm t-btn-primary" onclick="TMessage.success('保存成功'); closeTPopover('popover8')">保存</button>
      </div>
    </div>
  </div>
</div>

Popconfirm 确认框 ​

用于需要用户确认操作的场景。

⚠️确定要删除吗？
❓确定要退出登录吗？
html
<div class="t-popover t-popover-top t-popover--confirm" id="popconfirm1">
  <button class="t-btn t-btn-danger" onclick="toggleTPopover('popconfirm1')">删除</button>
  <div class="t-popover-content">
    <div class="t-popover-body" style="display: flex; align-items: center; gap: 8px;">
      <span style="color: var(--t-danger); font-size: 18px;">⚠️</span>
      <span>确定要删除吗？</span>
    </div>
    <div class="flex-gap" style="margin-top: 12px; justify-content: flex-end;">
      <button class="t-btn t-btn-sm" onclick="closeTPopover('popconfirm1')">取消</button>
      <button class="t-btn t-btn-sm t-btn-danger" onclick="TMessage.success('已删除'); closeTPopover('popconfirm1')">确定</button>
    </div>
  </div>
</div>

      <span>确定要退出登录吗？</span>
    </div>
    <div class="flex-gap" style="margin-top: 12px; justify-content: flex-end;">

  </div>
</div>

不同尺寸 ​

小尺寸弹出框
大尺寸弹出框
这是一个大尺寸的弹出框，可以容纳更多内容。适合显示详细的信息或复杂的操作。
html
<div class="t-popover t-popover-top" id="popover13">
  <button class="t-btn" onclick="toggleTPopover('popover13')">小尺寸</button>
  <div class="t-popover-content t-popover--small">
    <div class="t-popover-body">小尺寸弹出框</div>
  </div>
</div>
<div class="t-popover t-popover-top" id="popover14">
  <button class="t-btn" onclick="toggleTPopover('popover14')">大尺寸</button>
  <div class="t-popover-content t-popover--large">
    <div class="t-popover-title">大尺寸弹出框</div>
    <div class="t-popover-body">这是一个大尺寸的弹出框，可以容纳更多内容。适合显示详细的信息或复杂的操作。</div>
  </div>
</div>

Focus 触发 ​

输入提示信息
html
<div class="t-popover t-popover-top t-popover--focus" id="popover15">
  <input type="text" class="t-input t-p-5" placeholder="聚焦显示" style="width: 200px;" onfocus="openTPopover('popover15')" onblur="closeTPopover('popover15')">
  <div class="t-popover-content">
    <div class="t-popover-body">输入提示信息</div>
  </div>
</div>

  </div>
</div>

API ​

样式类 ​

类名说明
t-popover弹出框容器
t-popover-top上方弹出
t-popover-bottom下方弹出
t-popover-left左侧弹出
t-popover-right右侧弹出
t-popover--hoverHover 触发方式
t-popover--focusFocus 触发方式
t-popover--confirm确认框样式
t-popover--tbe硅胶质感样式
t-popover--delay延迟关闭样式
t-popover--small小尺寸内容
t-popover--large大尺寸内容
is-disabled禁用状态
t-popover-content弹出内容
t-popover-title标题
t-popover-body内容主体

JavaScript 方法 ​

方法名说明参数

openTPopover(id)打开弹出框id: 弹出框的 ID
closeTPopover(id)关闭弹出框id: 弹出框的 ID
