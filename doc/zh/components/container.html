API ​

Container 属性 ​

属性名类型默认值说明
classstring-容器类名，支持 t-container--vertical 垂直布局

Header 属性 ​

属性名类型默认值说明
classstring-顶栏类名，支持 t-header--sm、t-header--lg

Aside 属性 ​

属性名类型默认值说明
classstring-侧边栏类名，支持 t-aside--sm、t-aside--md、t-aside--lg、t-aside--xl

Main 属性 ​

属性名类型默认值说明
classstring-主内容区类名

Footer 属性 ​

属性名类型默认值说明
classstring-底栏类名
