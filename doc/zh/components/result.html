Result 结果 ​

Result 结果展示组件，用于显示操作结果，包括成功、警告、错误、信息等状态。

基础用法 ​

成功状态 ​

显示成功操作的结果反馈。

成功提示
请根据提示进行操作
html
<div class="t-result">
  <div class="t-result__icon t-result__icon--success">
    <svg viewBox="0 0 1024 1024" fill="currentColor">
      <path d="M512 64C264.6 64 64 264.6 64 512s200.6 448 448 448 448-200.6 448-448S759.4 64 512 64zm193.5 301.7l-210.6 292a31.8 31.8 0 0 1-51.7 0L318.5 484.9c-3.8-5.3 0-12.7 6.5-12.7h46.9c10.2 0 19.9 4.9 25.9 13.3l71.2 98.8 157.2-218c6-8.3 15.6-13.3 25.9-13.3H699c6.5 0 10.3 7.4 6.5 12.7z"/>
    </svg>
  </div>
  <div class="t-result__title">成功提示</div>
  <div class="t-result__subtitle">请根据提示进行操作</div>
  <div class="t-result__extra">
    <button class="t-btn t-btn-primary">返回</button>
  </div>
</div>

状态图标类型 ​

Result 组件内置了四种状态图标，通过 t-result__icon--{type} 修饰符类来实现：

状态修饰符类颜色值
成功t-result__icon--success#67C23A
警告t-result__icon--warning#E6A23C
错误t-result__icon--error#F56C6C
信息t-result__icon--info#909399

自定义内容 ​

自定义图标 ​

使用自定义图片作为图标。

抱歉，请求错误

  <div class="t-result__icon t-result__icon--custom">
    <img src="/demo-img/10-300x200.jpg" alt="404">

  </div>
</div>

CSS 类说明 ​

结构类 ​

类名说明
.t-result结果组件主容器
.t-result__icon图标容器
.t-result__title标题文本
.t-result__subtitle副标题文本
.t-result__extra额外操作按钮区域

状态修饰符类 ​

类名说明
.t-result__icon--success成功状态图标（绿色）
.t-result__icon--warning警告状态图标（橙色）
.t-result__icon--error错误状态图标（红色）
.t-result__icon--info信息状态图标（灰色）
.t-result__icon--custom自定义图标（用于图片）

布局类 ​

类名说明
.t-result-grid结果网格容器，用于展示多个结果卡片
