Cascader 级联选择器 ​

用于从一组相关联的数据集合中进行选择，例如省市区选择。

基础用法 ​

▼
指南 ▶
组件 ▶
html
<div class="t-cascader" id="cascader1">
  <input type="text" class="t-cascader__input" placeholder="请选择" readonly onclick="toggleCascader('cascader1')" id="cascader1-input">
  <span class="t-cascader__arrow">▼</span>
  <div class="t-cascader__dropdown">
    <div class="t-cascader__menu" id="cascader1-menu1">
      <div class="t-cascader__option" onclick="selectCascaderLevel1('cascader1', 'guide', '指南')">
        指南
        <span class="t-cascader__option-arrow">▶</span>
      </div>
      <div class="t-cascader__option" onclick="selectCascaderLevel1('cascader1', 'component', '组件')">
        组件
        <span class="t-cascader__option-arrow">▶</span>
      </div>
    </div>
  </div>
</div>

API ​

样式类 ​

类名说明
t-cascader级联选择器容器
t-cascader__input输入框元素
t-cascader__arrow下拉箭头
t-cascader__clear清空按钮
t-cascader__dropdown下拉菜单
t-cascader__menu菜单选项卡
t-cascader__option选项元素
t-cascader__option-arrow选项箭头
is-disabled禁用状态
is-active激活状态
is-medium中等尺寸
is-small小型尺寸
is-mini迷你尺寸

JavaScript 方法 ​

方法名说明参数
toggleCascader(id)切换级联选择器显示/隐藏id: 组件ID
selectCascaderLevel1(id, key, name)选择第一级选项id: 组件ID, key: 键名, name: 显示名称
selectCascaderLevel2(id, parentKey, key, name, fullPath)选择第二级选项id: 组件ID, parentKey: 父级键, key: 键名, name: 名称, fullPath: 完整路径
selectCascaderFinal(id, fullPath)最终选择id: 组件ID, fullPath: 完整路径
clearCascader(id, event)清空选择id: 组件ID, event: 事件对象
